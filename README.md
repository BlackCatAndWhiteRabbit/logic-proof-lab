# 命题逻辑证明实验室

这是一个面向离散数学命题逻辑证明训练的本地网页应用。用户可以输入目标定理，通过三条 Hilbert 公理、MP 规则和已经保存的定理逐步构造证明；系统会检查每一步是否合法，并用结构图展示证明过程和定理库依赖。

## 打开方式

方式一：直接用浏览器打开 `index.html`。

方式二：启动本地服务后访问 `http://127.0.0.1:5500`。

```bash
npm start
```

Windows 用户也可以双击 `run-site.bat`。

## 下载与源码

项目已同步到 GitHub，可直接下载或克隆使用：

git clone https://github.com/BlackCatAndWhiteRabbit/logic-proof-lab.git

## 支持语法

- `~P`：否定
- `P -> Q`：蕴含
- `P & Q`：合取
- `P | Q`：析取
- 支持括号与变量名，例如 `A`、`P1`、`Rain`
- 也兼容中文括号、中文句末标点，以及 `→`、`¬`、`∧`、`∨`
- 支持带前提的目标，例如 `A,B->(A->C)=>B->C`

## 内置公理

1. `A -> (B -> A)`
2. `(A -> (B -> C)) -> ((A -> B) -> (A -> C))`
3. `(~B -> ~A) -> (A -> B)`

## 功能

- 输入目标公式
- 添加公理实例
- 用 MP 规则从两个已有步骤推出新公式
- 保存已经完成的目标定理
- 手动逐层进入演绎子证明，使用临时假设栈
- 证明完成后显示绿色 `Accepted` 动画
- 保存定理时同时保存用户的证明步骤
- 一键生成定理库中“被引用定理指向后续定理”的依赖关系图
- 可视化展示 MP 推理依赖图
- 内置 `A -> A` 示例证明

## 演绎原理

目标可以写成最终定理。点击“进入一层演绎”时，系统只拆当前子目标最外层的 `->`：把前件压入临时假设栈，把后件作为新的子目标。保存定理时会保存当前用户证明步骤，并记录其中调用过的定理。
