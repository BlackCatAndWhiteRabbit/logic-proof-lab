公理与规则

  1. A -> (B -> A)
  2. (A -> (B -> C)) -> ((A -> B) -> (A -> C))
  3. (~B -> ~A) -> (A -> B)
  4. @x A -> A[x := t]
  5. @x (A -> B) -> (A -> @x B)
MP 规则 若已有 A 与 A -> B，则可推出 B。
UG 规则 若已有 A，且 x 不在当前假设中自由出现，则可推出 @x A。

证明步骤

每一步来自公理实例、临时假设、MP、UG 或定理库。

公式语法

  • @x A 表示全称量词
  • ?x A 表示存在量词
  • $\forall x P(x) \to Q(x)$ 这类 Markdown/LaTeX 输入会自动转换
  • P(x,y,z,w) 表示四元谓词
  • A | B 自动展开为 ~A -> B
  • A & B 自动展开为 ~(A -> ~B)
  • 暂不支持等词和函词
